# Deckology 🃏

Deckology es un motor de juego de cartas web diseñado para la gestión y personalización de zonas de juego, eventos y mazos. Este proyecto fue desarrollado como parte de mi formación en Desarrollo de Aplicaciones Web (DAW) y representa un reto técnico completo que combina frontend, backend y gestión de bases de datos.

## 🚀 Descripción del Proyecto

El objetivo de Deckology es permitir a los administradores gestionar de forma dinámica los componentes de un juego de cartas:
*   **Gestión de Zonas:** Creación y personalización de entornos (bosque, ciudad, desierto, etc.) con sus fondos e imágenes asociados.
*   **Sistema de Eventos:** Configuración de eventos que ocurren dentro de las zonas de juego.
*   **Interfaz de Administración:** Un panel de control intuitivo para gestionar toda la lógica del juego.

## 🛠️ Tecnologías Utilizadas

*   **Frontend:** HTML5, CSS3 (diseño responsivo, personalización de componentes), JavaScript (lógica de frontend y búsqueda).
*   **Backend:** PHP (arquitectura MVC - Modelo Vista Controlador).
*   **Base de Datos:** MySQL (gestión de zonas, eventos y perfiles de usuario).
*   **Servidor:** XAMPP / Apache.
*   **Versionado:** Git, Git LFS (para la gestión eficiente de activos multimedia de gran tamaño).

## ✨ Características Destacadas

*   **Arquitectura MVC:** Separación clara de responsabilidades para mejorar la mantenibilidad y escalabilidad del código.
*   **Diseño UI Profesional:** Interfaz de administración moderna, limpia y enfocada en la usabilidad, construida desde cero.
*   **Gestión de Activos:** Implementación de **Git LFS** para gestionar archivos multimedia pesados (música, imágenes de alta resolución) sin comprometer el rendimiento del repositorio.
